Detailed Description

Class that will handle all file read/write operations to a given medium on the sattelite.

Handles all operations related to files. Currently only operates on string data. Note: The LittleFileSystem is not compatible with PCs, however it offers enough beneifts for it to be worth it. Check the documentation for more information. TODO: Update to work with other types of data.
